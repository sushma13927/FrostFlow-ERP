❄️ FrostFlow ERP
Cold Storage Management System – User Guide

FrostFlow ERP is a web-based application designed to manage cold storage inventory, suppliers, and customer orders. The system helps track stored products, manage supply activities, and allows customers to place orders through an easy dashboard.

✨ Features

User Authentication – Secure login and registration system for users.

Role-Based Access – Separate dashboards for Inventory Manager, Supplier, and Customer.

Inventory Tracking – Monitor product quantity and weight stored in cold storage.

Product Management – Suppliers can add and manage their products.

Order Management – Customers can browse products and place orders.

Automated Billing – System automatically calculates total amount and GST.

Wallet System – Customers can maintain balance for transactions.

Notification System – Alerts users about order updates and low stock.

Responsive UI – Clean and modern interface built with Tailwind CSS.

Secure Data Storage – Uses SQLite relational database for reliable data management.

1️⃣ Software Required

Before running the project, install the following software:

1. Node.js

Download and install Node.js from:
https://nodejs.org

Install the LTS version.

2. Visual Studio Code (VS Code)

Download from:
https://code.visualstudio.com

VS Code will be used to open and run the project.

3. Web Browser

Use Google Chrome or any modern browser.

2️⃣ Download the Project

Open the project repository:

👉 https://github.com/sushma13927/FrostFlow-ERP

Then follow these steps:

Click the Code button

Select Download ZIP

Extract the ZIP file on your computer

You will get a folder named FrostFlow-ERP.

3️⃣ Open the Project in VS Code

Open Visual Studio Code

Click File → Open Folder

Select the FrostFlow-ERP project folder

Click Open

Now the project files will appear in VS Code.

4️⃣ Open the Terminal in VS Code

In VS Code menu click Terminal

Select New Terminal

A terminal window will open at the bottom of VS Code.

5️⃣ Install Required Packages

In the terminal, type the following command:

npm install

This command installs all the required dependencies for the project.

Wait until the installation finishes.

6️⃣ Run the Application

After installation, start the project using the following command:

npm run dev

This will start the FrostFlow ERP server.

7️⃣ Open the Application

Open Google Chrome

In the address bar type:

http://localhost:3000

Press Enter

The FrostFlow ERP Login Page will appear.

8️⃣ Create an Account

Click Register

Enter the following details:

Username

Email

Phone number

Password

Select your role:

Inventory Manager

Supplier

Customer

Submit the form.

9️⃣ Login to the System

Enter your username or email

Enter your password

Click Login

You will be redirected to your dashboard.
